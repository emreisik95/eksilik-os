# ek$ilik Support

ek$ilik is an independent, open-source Ekşi Sözlük client. It is not affiliated with or endorsed by Ekşi Sözlük.

## Get help

- Check that `https://eksisozluk.com` opens on the same network.
- For sign-in issues, open **Settings → Sign In** and complete the Ekşi Sözlük web sign-in flow.
- Offline topics and local filters can be managed from **Settings**.
- To report an app bug or request help, [open a GitHub issue](https://github.com/emreisik95/eksilik-os/issues/new).

Do not include passwords, session cookies, private messages, or other personal information in a support request.

## User-generated content

Content displayed by the app comes from Ekşi Sözlük. Use **Report entry** and **Block author** in an entry's action menu for moderation actions. Legal or service-level content requests should also be directed to Ekşi Sözlük through its official channels.
